│  │                                                                    │  │
│  │ Ejemplo Ej2: 0.25×1.0 + 0.25×1.0 + 0.35×1.0 + 0.15×1.0 = 1.0      │  │
│  │ Ejemplo Ej3: 0.25×0.0 + 0.25×0.0 + 0.35×0.0 + 0.15×0.0 = 0.0      │  │
